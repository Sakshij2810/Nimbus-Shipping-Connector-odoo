Nimbus Shipping Connector

Integrate Nimbus Shipping with Odoo

Features

  • Real-time shipping rates calculation
  • Automated label generation
  • Shipment tracking integration
  • Support for multiple packages
  • Sandbox mode for testing
  • Default package configuration
  • Service code management
  • Comprehensive error handling

Configuration

To configure the Nimbus Shipping Connector:

  1. Go to Inventory → Configuration → Delivery → Carriers
  2. Create or edit a carrier and select 'Nimbus Shipping' as delivery type
  3. Enter your Nimbus API credentials (available from your Nimbus account)
  4. Configure default package type and service code
  5. Save and test with a sales order
